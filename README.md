# Friends Pizza Hut - Fast Food Delivery System

## Project Structure

- backend/   → Node.js/NestJS REST API
- frontend/  → Flutter app (User, Admin, Rider panels)
- docs/      → Documentation, API specs, DB schema

## Features
- User, Admin, Rider panels
- Secure authentication (email/password, Google, Facebook, OTP)
- Dynamic, multi-city, scalable
- Real-time notifications, order tracking
- Modern, responsive UI/UX

## Tech Stack
- Backend: Node.js (NestJS/Express), PostgreSQL
- Frontend: Flutter (Android, iOS, Web)
- Auth: Firebase Auth
- Notifications: Firebase Cloud Messaging
- Payments: JazzCash, Easypaisa, Card
