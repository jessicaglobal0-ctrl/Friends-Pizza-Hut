# Friends Pizza Hut Frontend (Flutter)

This Flutter app supports Android, iOS, and Web from a single codebase.

## Features
- User, Admin, and Rider panels
- Modern, responsive, mobile-first UI
- Secure authentication (email/password, Google, Facebook, OTP)
- Real-time order tracking, notifications
- Dynamic content, multi-city support
- PKR currency, local payment integrations

## Setup
1. Install Flutter SDK
2. Run `flutter pub get` in this directory
3. Run `flutter run` for mobile or `flutter run -d chrome` for web

## Folder Structure
